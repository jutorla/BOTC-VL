# BOTC Storyteller

Herramienta de narrador para **Blood on the Clocktower** construida con React + TypeScript + Vite.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en el navegador.

## Build de producción

```bash
npm run build
```
